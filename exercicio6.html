Largest Number with N Digits
Given an integer n, return the largest number that contains exactly n digits.

Example:

For n = 2, the output should be solution(n) = 99.

Input/Output:

  • [execution time limit] 4 seconds (js)
  • [memory limit] 1 GB
  • [input] integer n
    Guaranteed constraints:
    1 ≤ n ≤ 9.
  • [output] integer
    The largest integer of length n.
Solution

function solution(n) {
  // Calculate the largest number with n digits
  let largestNumber = Math.pow(10, n) - 1;

  return largestNumber;
}

Explanation:

This JavaScript function, called solution, takes an integer n as input and returns the largest possible number with n digits.

  • Math.pow(10, n): This expression calculates 10 raised to the power of n. This results in a number that has n digits, where all digits are 0 except the most significant digit, which is 1.
  • Math.pow(10, n) - 1: Subtracting 1 from the previous result adjusts the value to the largest number with n digits, all of which are 9.
  • return largestNumber;: The function returns the largest possible number with n digits.

Por exemplo, se n for 3, a função calculará o maior número possível com 3 dígitos, que é 999. Se n for 4, o resultado será 9999, e assim por diante. Essa função é útil para calcular limites superiores com base no número de dígitos em problemas de programação ou cálculos matemáticos.

For example, if n is 3, the function will calculate the largest possible number with 3 digits, which is 999. If n is 4, the result will be 9999, and so on. This function is useful for calculating upper bounds based on the number of digits in programming problems or mathematical calculations.
