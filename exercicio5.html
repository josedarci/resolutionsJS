Sum of Digits
You are given a two-digit integer n. Return the sum of its digits.

Example:

For n = 29, the output should be solution(n) = 11.

Input/Output:

  • [execution time limit] 4 seconds (js)
  • [memory limit] 1 GB
  • [input] integer n
    A positive two-digit integer.
    Guaranteed constraints:
    10 ≤ n ≤ 99.
  • [output] integer
    The sum of the first and second digits of the input number.
Solution

function solution(n) {
  // Extrai o dígito das dezenas e o dígito das unidades
  let dígitoDezenas = Math.floor(n / 10);
  let dígitoUnidades = n % 10;

  // Calcula a soma dos dígitos
  let soma = dígitoDezenas + dígitoUnidades;

  return soma;
}

Explanation:

The solution function calculates the sum of the first and second digits of the given two-digit number.
